# Business Insights 360 – FMCG

## Overview
End-to-end Power BI project providing a 360° view of FMCG business performance across sales, finance, marketing, and supply chain.

## Tools Used
- Power BI
- SQL
- Excel
- DAX Studio

## Key Highlights
- Analyzed ₹3.7B+ net sales across multiple regions and customer segments
- Built 6+ interactive dashboards with YoY and monthly trend analysis
- Created and optimized 20+ DAX measures to improve report performance
- Enabled data-driven insights through KPIs, slicers, and drill-downs
